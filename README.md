# Random Quote Generator

This is a simple web application that generates random motivational quotes with a button click.

## 🚀 Features
- Generates random quotes
- Simple and clean UI
- Interactive button

## 🛠️ Tech Used
- HTML
- CSS
- JavaScript

## 📌 How to Use
1. Open the website
2. Click on "Get Quote"
3. Enjoy a new quote every time!
